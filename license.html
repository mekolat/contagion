Security, privacy

This website does not track or log any data on its users.

This website has a very strong Content Security Policy.

All requests are encrypted with TLS. This website has a Qualys SSL rating of A+.

This website uses HTTP Strict Transport Security (HSTS) with long duration.

This domain is protected by DNSSEC.
