Security

This website has a very strong Content Security Policy.

All requests are encrypted with TLS. This website has a Qualys SSL rating of A+.

This website uses HTTP Strict Transport Security (HSTS) with long duration.

This domain is protected by DNSSEC.



Privacy

This website, by itself, does not track or log any data on its users.

This website does not contain ads or any user behaviour tracking script (analytics).

The hosting provider may log non-personally-identifying information. See the GitLab Privacy Policy.

The CDN/proxy may log non-personally-identifying information. See the Cloudflare Privacy Policy.



Cookies

Cloudflare stores a _cfduid cookie in the browser, used for security purposes.



Data usage

Once this website is fully loaded, it does not maintain an active connection to the server, and does not download or upload files in the background.

This website caches itself in your browser, and registers a service worker.

The service worker allows you to access this website while being offline, as long as the data remains cached in your browser.

If you choose to install this website as a progressive web app, your browser may once in a while check for an updated version when you go online.



Transparency

This website is hosted on GitLab Pages.

This website is proxied through Cloudflare.
